#Automação de cotação PTAX
Script simples em Python pra buscar a cotação do dólar (USD) e euro (EUR) usando a API do BACEN e salvar em um CSV.

#O que ele faz:
-Puxa a cotação do dia
-Se não tiver (fim de semana/feriado), volta até 7 dias
-Salva tudo em cotacoes.csv
-Mostra no terminal

#Como rodar:
 rode o executavel

#Estrutura
CODE/
automacao.py
cotacoes.csv
README.md

Observação
os dados vêm da API oficial do Banco Central (PTAX).
